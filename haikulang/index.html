HaikuLang

A derivative of the Hector programming language that's structured to follow the haiku poem scheme.

More on Github Download

Try it yourself

helloworld.haiku 
Winter's biting cold... Winds howl 'hello world'... I stay warm in bed...
py haikulang helloworld.haiku

  1. Download the haikulang.py file from my repo

  2. Linux/Mac

    3. cd /path/to/haikulang.py py haikulang path/to/your/file.haiku

    Windows

    cd \path\to\haikulang.py py haikulang path\to\your\file.haiku
